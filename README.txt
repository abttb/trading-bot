
Trading Bot - Interactive Brokers Edition
----------------------------------------

תיק זה כולל את כל קבצי ההפעלה של הבוט:
- bot.py - הקובץ הראשי של הבוט
- run_bot.bat - קובץ להרצה אוטומטית (Windows)
- config.env - קובץ הגדרות עם מפתחות API

דרישות:
- Python 3.9 ומעלה
- TWS או IB Gateway פועלים ברקע עם חיבור ל־Interactive Brokers
- התקנת הספריות הדרושות בקובץ requirements.txt

שימוש:
1. פתח את run_bot.bat להפעלת הבוט.
2. ודא שהשוק פתוח והחיבור ל־IB פעיל.
